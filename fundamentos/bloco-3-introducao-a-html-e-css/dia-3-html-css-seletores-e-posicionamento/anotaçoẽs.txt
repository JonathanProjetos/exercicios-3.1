Agrupamento no css = itens que carregarem o mesmo atributo e class eles podem ser somado na css separando por 
vírgula e espaço.
pseudo classe utilizando houve = Quando passar o mause sobre o item destacado ele reagira deferente ex: li:hover quando passar
o mause sobre a lista poderá mudar de cor utilizando o background-color para a definição de cor. Obs; funicona somente 
com mause.
pseudo classe utilizando active =  quando selecionar algo com mause a cor interna do conteúdo pode reagir de forma diferente
quando ativado pode altera a cor do letreiro interno a sua escolha definindo color.
definição de pseudo-class= define por nome do item mais : e a ação houver,active.
